## 課題→機能 対応マインドマップ

```mermaid
mindmap
  root((😵 課題と対応機能))
    👶 初心者が動けない
      何から勉強すればいい
        🤖 学習ロードマップ自動生成
        📚 タスク別推薦教材
        🟡 今日やることリスト
      どの順番でやるか不明
        🗺️ フェーズ別タスク表示
        🟢 依存関係マップ
        🟡 マイルストーン表示
      受け身になってしまう
        ✅ 明確なタスクアサイン
        🟡 タスク期限リマインダー
        🟡 個人ビュー

    🔥 できる人が疲弊
      説明・設計の負担集中
        🤖 AIが技術スタックを提案
        🤖 タスク自動分解で説明不要に
        💬 質問テンプレで質問品質UP
      知識の属人化
        🗺️ ロードマップの文書化・共有
        📚 学習リソース一覧化
        🟢 カスタム教材追加
      レビュー時間がない
        ✅ チェックリスト付きタスク設計
        💬 コードレビュー依頼テンプレ
        🟢 ペアリング提案

    👥 チーム全体の問題
      進捗が見えない
        ✅ カンバンボード
        🗺️ チームビュー
        🟡 ガントチャート
        🟢 週次レポート
      役割分担が曖昧
        🏗️ ロール設定
        🤖 スキルベース担当割り振り
        🟢 負荷分散チェック
      要件定義できない
        📋 選択式ウィザード
        📋 機能チェックリスト
        🟡 MVP自動提案
```
