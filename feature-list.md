# 機能一覧 & 優先順位

> 優先度凡例：🔴 P0（MVP必須）／🟡 P1（早期追加）／🟢 P2（中期）／⚪ P3（将来）

---

## 🔐 1. 認証・ユーザー管理

| # | 機能名 | 詳細 | 優先度 | 理由 |
|---|--------|------|--------|------|
| 1-1 | メール＋パスワード登録 | 新規アカウント作成 | 🔴 P0 | 最低限の入り口 |
| 1-2 | ログイン / ログアウト | セッション管理 | 🔴 P0 | 必須 |
| 1-3 | Googleログイン（OAuth） | ソーシャルログイン | 🟡 P1 | サークル生はGoogle多用 |
| 1-4 | プロフィール設定 | 名前・アイコン・自己紹介 | 🟡 P1 | チームで誰か分かる |
| 1-5 | スキルレベル登録 | 得意技術タグ・習熟度入力 | 🔴 P0 | AI担当割り振りに必須 |
| 1-6 | 学習したい技術登録 | 目標スキル入力 | 🟡 P1 | 個人ロードマップ生成に使用 |
| 1-7 | パスワードリセット | メールで再設定リンク送付 | 🟢 P2 | あると安心、MVPは後回し可 |
| 1-8 | アカウント削除 | 退会処理 | ⚪ P3 | 法的要件だが急がない |

## 📋 3. 要件定義支援

| # | 機能名 | 詳細 | 優先度 | 理由 |
|---|--------|------|--------|------|
| 3-1 | プロダクト種別選択 | Web / Mobile / Game / AIなど選択 | 🔴 P0 | AI分析の前提情報 |
| 3-2 | 機能チェックリスト | ログイン・チャット・地図など選択式 | 🔴 P0 | 初心者でも入力できる仕組みの核心 |
| 3-3 | 難易度・規模設定 | 初心者向け / 学習用 / 実用 | 🔴 P0 | タスク粒度・技術選定に影響 |
| 3-4 | 自由記述入力 | 追加要件・参考URL・補足メモ | 🔴 P0 | チェックリストで拾えない要件を吸収 |
| 3-5 | 要件プレビュー・確認画面 | 送信前に内容確認 | 🟡 P1 | 誤送信防止 |
| 3-6 | 要件の編集・更新 | 後から要件を変更 | 🟡 P1 | 仕様変更は必ず起きる |
| 3-7 | MVP自動提案 | AIが「まずここまで」を提示 | 🟡 P1 | スコープ肥大化の防止 |
| 3-8 | 要件の曖昧さ指摘 | 不明確な記述をAIが検出 | 🟢 P2 | 質の高い要件定義に |

|---|--------|------|--------|------|

| 4-2 | タスク自動分解 | エピック→ストーリー→タスクに分解 | 🔴 P0 | 同上 |
